# Music Streaming en Solana

Proyecto desarrollado en Solana utilizando Anchor Framework que permite administrar una biblioteca de canciones mediante operaciones CRUD.

## Funcionalidades
- Crear streaming
- Agregar canciones
- Ver canciones
- Eliminar canciones
- Alternar disponibilidad

## Tecnologías
- Rust
- Solana
- Anchor
- TypeScript
